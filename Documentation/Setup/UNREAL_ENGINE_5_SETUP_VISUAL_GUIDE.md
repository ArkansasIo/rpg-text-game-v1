## ? COMMON ISSUES & FIXES

```
ISSUE: "Project won't open"
FIX:   Delete Intermediate & Binaries folders
       Right-click .uproject ? Generate VS files
       Try again

ISSUE: "Unreal Editor crashes on startup"
FIX:   Make sure Visual Studio is installed
       Make sure .NET 6.0+ is installed
       Update GPU drivers

ISSUE: "Can't find Widget Blueprint"
FIX:   Content Browser ? Settings (bottom right)
       Toggle "Show Engine Content"
       Toggle "Show Plugin Content"

ISSUE: "Project takes forever to load"
FIX:   First load is slow (normal)
       Wait 5-10 minutes
       Next loads will be faster

ISSUE: "Out of disk space"
FIX:   Unreal Engine is ~50 GB
       Make sure you have 60+ GB free
       Delete other projects if needed
```
